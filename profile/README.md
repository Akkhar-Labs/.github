# Akkhar-Labs | অক্ষর

**Building a complete computing ecosystem where language, software, and hardware are designed together.**

Akkhar-Labs is an independent research and engineering organization building AI infrastructure, developer tools, and computing systems from the ground up.

Modern computing is built around English-first assumptions. This creates barriers for billions of people who want to understand, build, and control the technology they use.

We are building alternatives — starting from Bangladesh — where computing is designed around human language, not the other way around.

---

## 🏛️ The Akkhar Stack

Akkhar-Labs develops a connected computing stack from hardware to user interface:

**Nirdeshika → BAPS → Matrika → Kotha**

### Nirdeshika — Instruction Architecture

A custom 16-bit instruction set architecture (ISA) designed as a foundation for Bengali-oriented computing environments. The ISA defines how software communicates with hardware using language-aware constructs.

**Status:** Nirdeshika-16 specification complete. Assembler (Boyon) and simulator (Shimu) are working.

### BAPS — System Kernel Layer

A lightweight kernel layer that connects hardware capabilities with software execution. Designed for minimal overhead and direct hardware access.

**Status:** Early architecture design. Research in progress.

### Matrika — Operating System

An operating system designed for language-native computing environments. Built on BAPS and Nirdeshika.

**Status:** Research and early architecture design.

### Kotha — Programming Language

A programming language that allows developers to express instructions in natural Bengali sentences, compiling directly to Nirdeshika machine code.

**Status:** Language design and compiler research in progress.

Together, these projects form a complete, native computing stack — designed to work together from the ground up.
